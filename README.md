# avue-data、dataV、avue搭建的大屏数据展示系统
```
1、无组件间交互
   -静态页面使用avue-data快速构建
   -导出后，拆分整合成.vue页面，参考src/views/main.vue
2、需要交互，比如点击切换数据、搜索等
   -图表使用dataV的echarts
   -页面元素使用avue或elementUI
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
